21. EventLoop解释
由于 JS 是门非阻塞单线程语言，JS 在执行的过程中会产生执行环境，这些执行环境会被顺序的加入到执行栈中。如果遇到异步的代码，会被挂起并加入到 Task（有多种 task） 队列中。一旦执行栈为空，Event Loop 就会从 Task 队列中拿出需要执行的代码并放入执行栈中执行。
```
所以正确的一次 Event loop 顺序是这样的：
① 执行同步代码，这属于宏任务
② 执行栈为空，查询是否有微任务需要执行
③ 执行所有微任务
④ 必要的话渲染 UI
⑤ 然后开始下一轮 Event loop，执行宏任务中的异步代码

```
